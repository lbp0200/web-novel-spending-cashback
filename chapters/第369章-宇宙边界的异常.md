# 第369章 宇宙边界的异常

银河系边境，星际观测站“黎明之眼”。

林雨桐站在观测大厅的全息投影前，眉头紧锁。作为林家第十七代传人，她如今已是维度守护者联盟的副指挥官，负责监察银河系周边维度的异常波动。

“报告长官，第七十三次预警。”副官的声音从身后传来，带着一丝难以掩饰的不安，“宇宙边界出现了新的异常反应。”

林雨桐没有回头，只是轻轻点了点头。她早就注意到了——过去三个月以来，宇宙边界的能量波动越来越频繁，越来越剧烈。那种波动的模式，她从未在任何文献中见过。

“投影放大一千倍。”她下达命令。

全息投影瞬间拉近，宇宙边界的景象清晰地呈现在眼前。那里是一片虚无与混沌的交界处，正常的宇宙空间在那里逐渐消散，化为最原始的能量尘埃。然而此刻，在那片虚无之中，却有一点光芒在不断闪烁。

那不是星光，也不是任何已知的能量辐射。那是一种有节奏的、仿佛心跳一般的脉动。

“调取历史数据对比。”林雨桐的声音平静，但手指却在微微颤抖。

很快，历史数据呈现在投影旁边。过去三个月的异常波动被绘制成了一条曲线。那曲线的形态，竟然与某种古老文献中记载的“虚空心跳”完全吻合。

不，不可能是虚空心跳。根据家族史料记载，虚空早在数万年前就被先祖林渊封印，如今应该处于永恒的沉睡之中。除非……

“报告！”一名年轻的观测员突然惊呼，“异常波动强度突然增加了三倍！”

林雨桐猛然抬头，只见那原本微弱的光点正在急剧膨胀，释放出令人心悸的能量波动。宇宙边界开始出现裂痕，那裂痕以肉眼可见的速度向内蔓延。

“立即启动一级警报！”林雨桐当机立断，“同时向维度守护者联盟总部发送紧急求援信号！”

“是！”

观测站内顿时忙碌起来，警报声响彻整个空间站。林雨桐深吸一口气，努力让自己冷静下来。她知道，现在任何慌乱都可能导致灾难性的后果。

与此同时，在银河系另一侧的林家主星上，林家现任家主林浩然正在召开家族会议。警报信号传来时，所有人的脸色都变了。

“虚空心跳？”林浩然的祖父林承远猛地站起身，“这不可能！当年林渊先祖以自身为代价封印虚空，那封印绝对不可能在这么短的时间内失效！”

“但数据不会说谎。”林浩然看着手中的全息报告，脸色凝重，“第七十三次预警，强度是三个月前的十倍。如果这种趋势继续下去，封印很可能在百年之内彻底崩溃。”

“百年？”林承远冷笑一声，“我看不需要那么久。你们看这个波动曲线，它正在加速。按照这个速度，可能连十年都撑不到。”

会议室陷入了死一般的寂静。

林浩然站起身，走到窗边，望着窗外繁华的星际都市，心中思绪万千。林家世代守护银河系，从先祖林渊开始，已经延续了数万年。无数代林家人为了维护宇宙的和平与稳定，付出了青春、热血甚至生命。

如今，又到了考验林家的时刻。

“立即召集所有维度守护者。”林浩然转过身，目光坚定，“同时，加派探测队前往宇宙边界，我们要第一时间掌握最准确的情报。”

“家主！”一名年轻的家族成员突然站起来，“我有一个提议。”

林浩然看向他：“说。”

“根据家族史料记载，先祖林渊在封印虚空时，留下了一个备用方案。如果封印出现松动，可以通过激活某种特殊的能量场来加固封印。但激活这个能量场需要一个条件。”

“什么条件？”

“需要林渊先祖的血脉亲自前往封印核心，用自身的能量激活能量场。”那年轻人顿了顿，“这意味着，只有林家嫡系传人才有这个资格。”

林浩然沉默了。这个方案的危险性不言而喻——前往封印核心，就意味着要直面虚空的力量。即使有先祖留下的保护，能有多少生还的希望，谁也说不准。

“还有别的办法吗？”他问。

没有人回答。

就在这时，会议室的大门被推开，一名少女走了进来。她看上去不过十八九岁，一头长发随意地披在肩上，眼神中带着与年龄不符的沉稳。

“爷爷，让我去吧。”林诗音——林家第十七代最小的传人——说道，“我是林渊先祖的血脉，这是我的责任。”

所有人的目光都集中在了她身上。

林浩然张了张嘴，想要拒绝，但看到孙女眼中的坚定，最终只能叹了口气：“诗音，你知不知道这意味着什么？”

“我知道。”林诗音微微一笑，“但林家的责任，从来就不是逃避，而是面对。先祖当年能够牺牲自己封印虚空，我作为他的后人，至少应该去尝试。”

会议室再次沉默。

良久，林承远缓缓开口：“这孩子，有她先祖的风范。”

林浩然看着孙女，终于点了点头：“好。但你不能一个人去，我会让林雨桐的观测队护送你。同时，我们会召集联盟所有力量，做好最坏的打算。”

“我明白了，爷爷。”林诗音点头，“那我这就去准备。”

说完，她转身离开了会议室。

望着孙女离去的背影，林浩然心中百感交集。林家世代守护银河系，从来不缺少愿意为家族牺牲的人。但作为长辈，他多么希望这样的牺牲永远不要发生。

然而，命运的车轮已经开始转动。

在遥远的宇宙边界，那心跳一般的脉动依然在继续。而随着它每一次跳动，都有一丝微弱的能量从封印的裂缝中泄露出来，向宇宙深处蔓延。

那能量中，似乎蕴含着某种古老而危险的信息……

——本章完——
